## 遇到的问题

1. **pip editable install EXE 找不到 src 模块**：Windows 上 pip 的 zipapp .exe 启动器无法正确处理 editable install 的 .pth 文件。解决方案：将 src 布局改为 flat layout，重命名 src/ → hirehive/，并在 pyproject.toml 中显式配置 packages.find。
2. **DeepSeek thinking block**：DeepSeek V4 Pro 的响应包含 thinking block（非 text block），原 _collect_stream 未处理导致流式解析崩溃。解决方案：在 BaseAgent 中添加 thinking/redacted_thinking/thinking_delta/signature_delta 事件处理。
3. **Windows GBK 编码**：中文输出乱码。解决方案：main() 中 reconfig stdout/stderr 为 utf-8，测试时设置 PYTHONIOENCODING=utf-8。
