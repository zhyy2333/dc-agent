# 开发需求文档

## 项目概述

**HireHive — 全流程 AI 求职 Agent**，覆盖求职全生命周期：岗位发现 → 简历匹配 → 自动投递 → 面试准备 → Offer 对比。

## 核心需求

### 1. 岗位发现 (Boss Search)

- 在 BOSS 直聘上按关键词、城市、薪资范围自动搜索
- 支持翻页、无限滚动加载
- 抓取岗位详情（描述、要求、标签、薪资）
- 去重存入本地数据库

### 2. 简历匹配 (Resume Match)

- 上传 PDF/DOCX 简历，LLM 提取结构化信息
- 五个维度打分：技能、经验、学历、薪资、综合印象
- 输出 S/A/B/C/D 评级和匹配理由
- 过滤低分岗位

### 3. 自动投递 (Auto Apply)

- 在 BOSS 直聘岗位详情页点击"立即沟通"
- LLM 生成个性化打招呼消息
- 支持附件简历上传
- 投递前必须用户确认

### 4. 面试准备 (Interview Coach)

- 根据 JD + 简历生成 10-15 道个性化面试题
- 覆盖技术题、项目题、行为题
- 交互式模拟面试，AI 追问并逐题评分
- 导出面试准备报告

### 5. Offer 对比 (Offer Adviser)

- 五个维度对比：薪资、发展、生活、公司、文化
- 加权评分排名
- 优劣势分析和风险提示

## 非功能需求

- Windows/macOS/Linux 跨平台
- CLI + 交互式 REPL 两种模式
- SQLite 本地持久化，无需外部服务
- 反爬策略：随机延迟、真实 UA、Session 持久化
- DeepSeek V4 Pro 作为 LLM 后端（通过 Anthropic SDK proxy）
